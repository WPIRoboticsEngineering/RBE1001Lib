14 /*
15  * The procedure is to send a 10us pulse on the trigger line to
16  * cause the ultrasonic burst to be sent out the speaker. When
17  * that happens, the echo line will go high. When the echo from
18  * the sound returns, the echo line will go low. This pulse length
19  * is measured using the pulseIn function that returns the round-trip
20  * time in microseconds which is then converted to centimeters and
21  * returned.
22  * The timer interrupt routine will send the 10us pulse every 100ms and
23  * the sensor ISR will time from the rising edge to the falling edge.
24  * This time is saved in "roundTripTime" and used to compute the distance.
25  */
